## 产品名字：WordFriend

## 产品描述：该产品旨在帮助英语学习困难的朋友更好的学习英语、背诵单词。可以输入文章，产品可以帮助你统计单词出现的频率并记录。可以通过修改生疏度来对单词进行归类，方便背诵。
